# HumanRobot_Aster

Ich fand diesen Roboter https://www.thingiverse.com/thing:3992150 und nun ist er die grundlage für dieses Repo.
Der Roboter Aster hat 20 DoF, Bestehend aus : Kopf 2, Arm je 4, Hand je 2, Bein je 3.
In meinen geänderten Stl datein befindet sich ein Face mit 2 plätzen für OLed 1306 und eine PiCam.
Ich betreibe ihn mit einem Raspberry Pi4 sowie 2 Pca9685 für die Servos.
Ros2 wird später hinzugefügt.
